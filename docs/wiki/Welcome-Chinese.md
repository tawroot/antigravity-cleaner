# 🇨🇳 Antigravity Cleaner Pro - 中文指南

由 **Tawana Network** 提供的专业系统清理和优化工具。

## 🌟 核心功能
- **真正的便携版:** 无需安装，数据保存在本地 `data` 文件夹中。
- **Apple 风格界面:** 现代且响应式的设计。
- **多浏览器支持:** 支持 8 种以上浏览器的会话管理。

## 🚀 快速安装
在 macOS 或 Linux 终端中运行以下命令：
`curl -sL https://raw.githubusercontent.com/tawroot/antigravity-cleaner/main/install.sh | bash`

## 🛡️ 安全与隐私
所有清理和备份操作均在本地完成。您的数据绝不会上传到任何服务器。会话备份使用 AES-256 加密。
