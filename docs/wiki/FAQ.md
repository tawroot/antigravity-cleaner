# FAQ - Frequently Asked Questions

## General

### What is Antigravity Cleaner?
A professional utility to clean Antigravity IDE traces, manage browser sessions, and fix Google connectivity issues.

### Is it free?
Yes, 100% free for personal use.

### Is it safe?
Absolutely. The "Analyze" function is read-only. All changes require confirmation.

### What platforms are supported?
- Windows 10/11 (x64)
- macOS Catalina 10.15+
- Linux (Ubuntu, Fedora, Arch)

## Troubleshooting

### How to fix 403 Forbidden error?
1. Open Antigravity Cleaner
2. Go to **Network** tab
3. Click **Flush DNS**
4. Click **Reset Network**
5. Restart your IDE

### How to fix region lock?
Use the **Google Test** page to diagnose which services are blocked. For region locks, you may need a VPN.

### Login stuck on "Setting up account"?
1. Run Quick Clean from Cleaner tab
2. Flush DNS from Network tab
3. Clear browser cache
4. Try a personal Gmail instead of Workspace

### Does it work in Iran/China/Russia?
Yes! Designed for restricted regions. The tool works offline - no internet required for scanning.

## Technical

### Where are sessions stored?
Sessions are stored locally in encrypted format:
- **Windows:** `%APPDATA%\AntigravityCleaner\sessions\`
- **macOS:** `~/Library/Application Support/AntigravityCleaner/sessions/`
- **Linux:** `~/.config/antigravity-cleaner/sessions/`

### What data is collected?
None. Zero telemetry. Everything stays on your device.

### Can I use it commercially?
No. Personal use only. See [LICENSE](https://github.com/tawroot/antigravity-cleaner/blob/main/LICENSE).
